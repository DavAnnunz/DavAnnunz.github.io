Description

The first challenge I chose to test myself is the creation of a mobile application. It is a game based on the rotation of an object around a sphere with the aim of collecting points within the end of the time. It moves forward during the game and the player has to rotate the world sphere to hit the plus objects and avoid the negative ones. There are two way to control the game, one based on two buttons to turn to rigth or left, the other one use a right or left swipe to rotate the sphere.

My goals with this project:

I started this project to improve my knowledge about the 3D environment using 3D models, I especially tried to use quaternion and interpolation.
